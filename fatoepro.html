Os Três Produtos Notáveis

São multiplicações algébricas que aparecem tanto que ganharam "fórmulas rápidas". Decorar isso poupa tempo valioso.

Quadrado da Soma:

(a + b)² = a² + 2ab + b²

Quadrado da Diferença:

(a - b)² = a² - 2ab + b²

Soma pela Diferença:

(a + b)·(a - b) = a² - b²
O PECADO CAPITAL DA ÁLGEBRA:
Achar que (a + b)² é igual a a² + b². ISSO É UM CRIME MATEMÁTICO! Você NUNCA pode esquecer o termo do meio (+ 2ab).
Caminho Inverso

Fatoração: Desfazendo o Produto

Fatorar é transformar uma soma/subtração algébrica em uma multiplicação. É fazer o caminho de volta.

1. Fator Comum em Evidência: Puxe para fora quem se repete.
Ex: ax + ay = a(x + y)
2. Agrupamento: Evidência dupla, feita em "grupinhos".
Ex: ax + bx + ay + by = x(a+b) + y(a+b) = (a+b)(x+y)
3. Diferença de Quadrados: O caminho inverso de (a+b)(a-b).
Ex: x² - 25 = (x + 5)(x - 5)
