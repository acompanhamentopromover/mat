📏 1. O que é uma Grandeza?

Tudo o que se pode medir

Em matemática e física, uma Grandeza é tudo aquilo que pode ser medido, contado ou pesado.

Exemplos: Tempo (horas), Velocidade (km/h), Distância (metros), Preço (euros/reais), Número de operários, Quantidade de fatias de pizza.

Emoções como "Alegria" ou "Amor" NÃO são grandezas porque não as podemos medir numa balança!

📈 2. Grandezas Diretamente Proporcionais (GDP)

As Duas Sobem Juntas!

Duas grandezas são Diretamente Proporcionais quando o aumento de uma provoca o aumento da outra na mesma proporção (ou se uma diminui, a outra diminui igualmente).

O Gráfico de uma GDP é sempre uma Linha Reta que passa pela origem (0,0).

O Exemplo do Supermercado:
Se 1 maçã custa R$ 2,00...
Se eu comprar o dobro (2 maçãs), vou pagar o dobro (R$ 4,00).
A divisão (razão) entre elas é sempre constante: 2/1 = 4/2 = 2.
📉 3. Grandezas Inversamente Proporcionais (GIP)

A Gangorra: Uma Sobe, Outra Desce

Duas grandezas são Inversamente Proporcionais quando o aumento de uma provoca a diminuição da outra na mesma proporção.

O Gráfico de uma GIP é sempre uma curva chamada Hipérbole.

O Exemplo da Viagem:
Para ir a uma cidade a 50 km/h, levo 2 horas.
Se eu for o dobro mais rápido (100 km/h), o tempo da viagem cai pela metade (1 hora).
A multiplicação (produto) entre elas é sempre constante: 50 × 2 = 100 × 1 = 100.
✖️ 4. Como Resolver? (Regra de Três Simples)

O Truque das Setas

Para resolver problemas de proporção, montamos as frações lado a lado e usamos um truque visual para saber como multiplicar:

SE FOR DIRETA (GDP)

2x
Multiplica em CRUZ ("X") 1·x = 2·2 ➔ x = 4
