📐 1. Forma Geral

A estrutura da equação

Toda equação do 2º grau pode ser escrita na forma ax² + bx + c = 0, com a ≠ 0. Ela é dita incompleta quando um dos coeficientes b ou c é igual a zero (ou ambos).

  • a é o coeficiente do termo quadrático (nunca pode ser zero).
  • b é o coeficiente do termo linear.
  • c é o termo independente (constante).
  • Incompleta: quando b = 0 ou c = 0 (ou ambos).
🔍 2. Tipo ax² + bx = 0 (falta c)

Colocar em evidência

Quando a equação não tem termo independente (c = 0), a resolução é simples: colocamos x em evidência e usamos a regra do produto nulo.

Exemplo: 3x² - 6x = 0 → 3x(x - 2) = 0 → x = 0 ou x = 2. S = {0, 2}.
⚡ 3. Tipo ax² + c = 0 (falta b)

Isolar x² e raiz quadrada

Quando b = 0, isolamos x² e depois extraímos a raiz quadrada. Cuidado com o sinal!

Exemplo: 2x² - 8 = 0 → 2x² = 8 → x² = 4 → x = ±2. S = {2, -2}.
Atenção: x² + 9 = 0 → x² = -9 → sem solução real (∅).
🎯 4. Caso ax² = 0

A única raiz dupla

Se b = 0 e c = 0, a equação reduz-se a ax² = 0. Como a ≠ 0, temos x² = 0, logo a única solução é x = 0 (raiz dupla).

Exemplo: 5x² = 0 → x = 0 (duas raízes iguais a zero).
